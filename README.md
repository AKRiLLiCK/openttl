## <img src="https://api.iconify.design/lucide:cpu.svg?color=%23A31F34" width="28" height="28" align="top" /> Mission
**OpenTTL** is an open-design repository providing high-fidelity, datasheet-compliant TTL/LS logic components for **Logisim Evolution**. 

Standard simulator libraries often rely on abstract black-box components that mask physical behavior. OpenTTL prioritizes structural integrity—every component is built from first principles (gates, flip-flops, multiplexers) to ensure 1:1 simulation fidelity, including timing, high-impedance (High-Z) states, and standard DIP-16 physical mapping.

## <img src="https://api.iconify.design/lucide:terminal.svg?color=%2328A745" width="28" height="28" align="top" /> Quick Start
1. **Importing**: Load any component into Logisim Evolution via `Project > Load Library > Logisim Library...`
2. **Implementation**: All components follow standardized physical DIP-16 pinouts for the external appearance, ensuring they function as drop-in hardware blueprints.
3. **Verification**: Each component is verified against standard timing diagrams rather than static truth tables to account for clock-edge triggering.
